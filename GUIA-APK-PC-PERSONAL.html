Fit Match en tu PC personal

Guia para trabajar el proyecto y generar el APK desde casa, sin las restricciones del trabajo.

Lo primero: tranquilidad

El Fit-Match.apk que ya tienes es un envoltorio de tu pagina web. Cuando actualizas el juego y lo subes a GitHub, la app se actualiza sola. Solo hay que regenerar el APK si cambias el nombre, el icono o la identidad de la app.

Parte A - Trabajar el codigo del juego

  1. Instala Git / GitHub Desktop

    Baja GitHub Desktop de desktop.github.com e inicia sesion con tu cuenta de GitHub (la misma: joshizaguirrea-hub).

  2. Clona el proyecto

    En GitHub Desktop: File → Clone repository → elige joshizaguirrea-hub/fit-match-game. Se descarga a tu PC.

  3. Edita con tu asistente (Cascade)

    Abre la carpeta en tu editor. Cuando cambies algo, en GitHub Desktop das Commit y luego Push. GitHub Pages publica solo tu sitio.

Parte B - Generar el APK (PWABuilder)

  1. Abre PWABuilder

    Ve a pwabuilder.com y pega tu URL:
    https://joshizaguirrea-hub.github.io/fit-match-game/

  2. Genera el paquete Android

    Package For Stores → tarjeta Android → Generate Package. En Signing key deja New. Descarga el ZIP.

  3. Descomprime el ZIP

    Adentro encontraras un .apk (viene unsigned = sin firmar) y un .aab (ese es solo para la Play Store). Guarda tambien el signing.keystore si aparece.

Parte C - Firmar el APK (para instalarlo)

Android no instala APKs sin firma. En tu PC personal esto es facil porque no hay proxy:

  1. Instala Java

    Baja Temurin JRE 17 de adoptium.net (o busca "Adoptium Temurin"). Instalador normal de doble clic. Marca la opcion de "Add to PATH" si la ofrece.

  2. Baja el firmador

    Descarga uber-apk-signer-1.3.0.jar desde:
    github.com/patrickfav/uber-apk-signer/releases (boton Assets). Ponlo en la MISMA carpeta que tu APK.

  3. Firma con un comando

    Abre una terminal (CMD/PowerShell) en esa carpeta y ejecuta:

    java -jar uber-apk-signer-1.3.0.jar --apks "Fit Match-unsigned.apk"

    Te crea un archivo ...-aligned-debugSigned.apk → ESE es el instalable.

Parte D - Instalar en el celular (Android)

  1. Pasa el APK al celular

    Por WhatsApp (a ti mismo), correo o cable USB.

  2. Abrelo e instala

    Android avisara "fuente desconocida" → Ajustes → Permitir → Instalar. Es normal, es tu propia app.

Notas honestas

  • Solo Android instala APKs. En iPhone es "Compartir → Anadir a inicio" desde Safari.
  • La app necesita internet (carga tu web en vivo) y se actualiza sola.
  • La firma "debug" sirve perfecto para compartir con amigos/familia. Para la Play Store se usa otra firma.
